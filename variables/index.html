JavaScript Revisit: JS Variables

Variables can be declared in 4 ways

  • Automatically
  • using var, old version. It has global scope. Use it if you must support old browser
  • using let, it has block scope, must be declared before use and can not be redeclared in the same scope; however, can be reassign. Let does not bind to this and are not hoisted.
  • using const, which have block scope. It cannot be redeclared and Reassigned. It can not be reassign, does not bind to this, and are not hoisted. Special note: you can change the elements of constant array, and change the properties on constant object.
