JavaScript Identifires/Name

→ Identifiers are JavaScript names. A JavaScript name must begin with the following:

  • A letter (A-Z or a-z)
  • A dollar sign ($)
  • or an underscore (_)

→ Case Sensitive All JavaScript identifiers are case sensitive. lastName and lastname are not same

→ Lower Camel case is the best policy for naming convention

  • firstName
  • interCityName
