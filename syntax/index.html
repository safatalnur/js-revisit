Javascript Revisit: JS Syntax

Javascript syntax is the set of rules.

JavaScript syntax defines two types of values:

→ Fixed Values are called Literals. For example, Numbers and Strings

  • 10.50
  • "John doe"

→ Variables are used to store data values. An equal sign is used to assign values to variables.

  • let x;
  • x = 6;

JavaScript Expressions are a combination of values, variables, and operators, which computes to a value. the computation is called an evaluation.
